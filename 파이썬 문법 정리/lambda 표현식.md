## lambda 표현식

`일회용 익명 함수` 라고 기억

```
def sum(x, y):
	return x+y
```

```
lambda x, y : x+y
```

> 위와 아래의 내용은 동일하다.

- 차이점 : 람다는 함수명이 없는 익명함수(자바스크립트에서의 익명함수와 유사). 즉 이름이 없기에 호출이 불가능. 그래서 변수를 선언해서 호출할수 있음.

### 정렬에서의 유용성

```
arr=sorted(arr,key = lambda key:key[1])
```

- 위의 식은 리스트로 된 요소들을 1번째 요소를 통해서 정렬하기 위한 코드
